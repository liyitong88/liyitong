# 医学装备科任务闭环管理系统使用说明

## 一、系统用途

本系统用于医学装备科日常事务管理，主要用于：

1. 设备维修登记
2. 设备采购登记
3. 计量校准登记
4. 设备验收登记
5. 质控问题跟进
6. 任务闭环与历史追溯
7. 月度工作统计与导出

适合手机和电脑使用，登录后即可开始录入和跟进任务。

## 二、登录方式

系统默认账号：

- 账号：`admin`
- 密码：`admin123`

登录后可进入首页查看待办、任务列表、统计报告和审计记录。

## 三、首页怎么看

首页主要显示四类信息：

1. 今日提醒
2. 即将提醒
3. 超期任务
4. 今日待办

首页还会展示：

- 优先提醒任务
- 今日待办任务
- 最近操作记录

点击任意统计卡或任务条目，都可以直接进入对应任务范围或任务详情。

## 四、如何新增任务

进入“新增任务”后，按顺序填写以下内容：

- 任务标题
- 任务类型
- 涉及科室
- 设备名称
- 设备编号
- 负责人
- 截止日期
- 提醒日期
- 问题描述

建议尽量一次填完整，这样后续查找和统计更方便。

新增任务时，系统支持：

- 直接保存
- 保存并继续新增下一条
- 套用常用模板

## 五、如何跟进任务

打开任务详情后，可进行以下操作：

- 查看任务信息
- 添加处理记录
- 上传附件或图片
- 修改任务状态
- 复制任务编号
- 复制完整摘要
- 复制为新任务
- 查看上一条和下一条任务

建议每次处理后都补一条记录，例如：

- 已联系厂家
- 工程师已到场
- 已更换配件
- 已完成验收

这样可以形成完整闭环。

## 六、如何完成闭环

任务处理完成后，在任务详情中点击“完成闭环”，填写：

- 完成时间
- 处理结果
- 是否恢复正常
- 费用金额
- 维修单位
- 验收人
- 完成备注

完成后任务状态会变为“已完成”。

已完成或已取消的任务默认不建议随意删除，便于后续追溯。

## 七、如何查看历史记录

系统会自动记录以下内容：

- 任务创建
- 任务修改
- 状态变更
- 处理记录
- 附件上传
- 完成闭环

在“审计”页面可以按时间、类型、操作人进行筛选，适合查历史和追踪过程。

## 八、如何生成报告

在“报告”页面可生成：

- 本月工作报告
- 本周工作报告
- 自定义时间报告

报告会统计：

- 新增任务数量
- 完成任务数量
- 未完成任务数量
- 超期任务数量
- 维修类任务数量
- 采购类任务数量
- 科室分布
- 费用合计
- 重点事项列表

支持导出 Excel，也可以复制报告正文用于汇报。

## 九、如何导出和备份

系统支持：

- 导出任务列表
- 导出审计记录
- 导出月报
- 导出完整备份
- 导入备份恢复数据

建议定期导出备份，避免浏览器清理数据后丢失记录。

## 十、手机上怎么用

手机和电脑连同一个 Wi-Fi 时，可以直接在手机浏览器打开电脑提供的地址。

如果希望像 App 一样使用，可以在手机浏览器中选择：

- 添加到主屏幕
- 安装应用

这样打开会更方便。

## 十一、常见问题

### 1. 忘记密码怎么办？

默认管理员账号为 `admin / admin123`。如后续修改过密码，请联系管理员。

### 2. 数据会不会自动同步到别的手机？

当前版本默认保存在浏览器本地，不会自动同步。换设备时请先导出备份，再导入到新设备。

### 3. 为什么建议经常备份？

因为本版本是轻量化本地管理模式，浏览器清理缓存或换设备时，数据需要靠备份恢复。

### 4. 手机打不开怎么办？

请确认：

1. 手机和电脑是否连接同一个网络
2. 电脑地址是否输入正确
3. 防火墙是否阻止了访问

## 十二、使用建议

为了让系统更适合科室日常使用，建议：

1. 每天先看首页“今日待办”
2. 处理完任务及时补记录
3. 任务完成后务必做闭环
4. 每周或每月导出一次备份
5. 统计报告用于月度汇报
